HTML

Es la forma como se organiza o se estructura la información en una página web. Es la que ordena todo el contenido desde la ubicación del título hasta los pié de página.

CSS

Es el lenguaje con el que puedes crear estilos al contenido de la página, de esta manera la página no se verá como letras una encima de la otra, sino con cuadros, estilos, diagramas y demás cosas agradables ante el ojo humano, que la hace ser atractiva.

JS

Este lenguaje te permite crear una interactividad con el usuario que está visitando tu página, donde puedes pedir datos, vistas dinámicas, pregunta y respuesta y demás.
